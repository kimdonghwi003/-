# PRD (Product Requirements Document)
# AI 기반 보컬 트레이닝 플랫폼

---

## 1. 제품 개요

### 1.1 서비스명 (가칭)
**VocalAI** – AI 기반 보컬 정밀 분석 및 트레이너 매칭 플랫폼

### 1.2 서비스 목적
사용자의 음성 파일을 AI로 분석하여 개인 맞춤형 보컬 피드백을 제공하고, 분석 결과를 바탕으로 실력에 맞는 곡 추천 및 MR 제작, 그리고 전문 보컬 트레이너와의 온라인 레슨 매칭까지 원스톱으로 지원하는 플랫폼.

### 1.3 핵심 가치
- **접근성**: 로그인 없이도 음성 분석 요청 가능 (학생 비회원 지원)
- **정밀성**: AI 기반 음정·박자·성량·음색 다차원 분석
- **연결성**: AI 분석 결과와 연동된 보컬 트레이너 추천 및 매칭

---

## 2. 사용자 계정 구조

### 2.1 계정 유형

| 구분 | 설명 | 로그인 필요 여부 |
|------|------|----------------|
| **비회원 학생** | 음성 파일 업로드 및 요구사항 제출 가능, 기본 분석 결과 열람 | ❌ 불필요 |
| **회원 학생** | 비회원 기능 + 분석 히스토리 저장, 곡 추천, MR 제작, 레슨 예약/결제 | ✅ 필요 |
| **보컬 트레이너** | 학생 분석 결과 열람, 레슨 수락/거절, 스케줄 관리, 정산 | ✅ 필요 (별도 계정) |

### 2.2 계정 분리 원칙
- 학생 계정과 트레이너 계정은 **완전히 분리된 회원가입/로그인 플로우**를 가짐
- 트레이너는 별도 **심사 및 승인 프로세스**를 거쳐 활성화
- 하나의 이메일로 두 역할을 동시에 보유할 수 없음

## 3. 핵심 기능 명세

### 3.1 AI 기반 보컬 정밀 분석 및 개인 맞춤 곡 추천

#### 3.1.1 보컬 데이터 분석
- **대상 사용자**: 비회원 학생 포함 전체
- **입력**: 음성 파일 (MP3, WAV, M4A 등) + 요구사항 텍스트
- **분석 항목**:
  - 음정 (Pitch Accuracy)
  - 박자 (Rhythm / Timing)
  - 성량 (Volume Dynamics)
  - 음색 (Timbre / Tone Quality)
- **출력**: 시각화된 분석 리포트 (그래프, 점수, 피드백 텍스트)
- **비회원 제한**: 분석 결과는 1회성 열람, 저장 불가

#### 3.1.2 스마트 곡 추천
- **대상 사용자**: 회원 학생
- **기준**:
  - 분석된 음역대 (최저음 ~ 최고음)
  - 선호 장르 및 취향 (회원가입 시 입력)
- **추천 목록 구성**:
  - ✅ 현재 실력으로 완창 가능한 곡
  - 🔥 도전해볼 만한 곡 (실력 향상 동기 부여)

---

### 3.2 사용자 맞춤형 연습 도우미 (Key 조절 및 MR 제작)

#### 3.2.1 키(Key) 조절
- **대상 사용자**: 회원 학생
- **기능**: 사용자가 원하는 최적 음역대에 맞춰 MR의 키를 ±N 반음 조정
- **지원 범위**: -6 ~ +6 반음 (semitone)

#### 3.2.2 AI 보컬 제거 (MR 자동 생성)
- **대상 사용자**: 회원 학생
- **기능**: 원곡 파일 업로드 → AI가 보컬 트랙 분리 → 연습용 MR 생성
- **저작권 안내**:
  - 생성된 MR은 개인 노래 연습 목적으로만 사용 가능
  - 외부 유출 및 상업적 이용 엄격 금지
  - 저작권 가이드라인 동의 후 사용 가능

---

### 3.3 온라인 보컬 트레이너 매칭

#### 3.3.1 전문가 매칭
- **대상 사용자**: 회원 학생
- **방식**: AI 분석 결과(보완 필요 항목: 고음 처리, 호흡, 스케일 등)를 기반으로 적합한 트레이너 자동 추천
- **트레이너 프로필 표시 항목**:
  - 전문 분야, 지도 경력, 수강 후기, 레슨 가격

#### 3.3.2 온라인 레슨 예약 및 결제
- **대상 사용자**: 회원 학생, 보컬 트레이너
- **기능**:
  - 1:1 맞춤형 온라인 레슨 예약
  - 시간·장소 제약 없는 비대면 레슨
  - 인앱 결제 및 정산 시스템
  - 레슨 완료 후 리뷰 작성

## 4. 비회원 학생 상세 플로우

```
[홈 화면 접근]
    ↓
[음성 파일 업로드]
    ↓
[요구사항 입력] (선택 사항: 개선하고 싶은 부분, 목표 곡 등)
    ↓
[이메일 또는 링크로 결과 수령] (선택)
    ↓
[AI 분석 리포트 열람]
    ↓
[회원가입 유도 CTA] → 히스토리 저장 / 곡 추천 / MR 제작 / 레슨 예약
```

## 5. 트레이너 계정 주요 기능

| 기능 | 설명 |
|------|------|
| 프로필 관리 | 전문 분야, 경력, 가격, 자기소개 등록 |
| 매칭 수신 | 학생 AI 분석 결과와 함께 레슨 요청 수신 |
| 스케줄 관리 | 가능한 레슨 시간 설정 및 예약 확인 |
| 레슨 진행 | 온라인 화상 레슨 또는 피드백 메모 전달 |
| 정산 | 완료된 레슨 기준 수수료 차감 후 정산 |

---

## 6. 기술 요구사항 (개요)

| 항목 | 내용 |
|------|------|
| 음성 분석 | AI 오디오 분석 API (자체 또는 외부 연동) |
| 보컬 분리 | Spleeter, Demucs 등 음원 분리 오픈소스 또는 API |
| 키 조절 | 오디오 피치 쉬프팅 라이브러리 (예: Librosa, Rubberband) |
| 결제 | 국내 PG 연동 (예: 토스페이먼츠, 포트원) |
| 인프라 | 클라우드 기반 (AWS / GCP), 음성 파일 스토리지 |
| 보안 | 비회원 업로드 파일 일정 기간 후 자동 삭제 |

---

## 7. 비기능 요구사항

- **응답 속도**: 음성 분석 결과 제공까지 최대 2분 이내
- **파일 크기 제한**: 업로드 파일 최대 50MB
- **지원 포맷**: MP3, WAV, M4A, OGG
- **비회원 파일 보관**: 분석 완료 후 24시간 내 자동 삭제
- **모바일 최적화**: 반응형 웹 또는 모바일 앱 지원
