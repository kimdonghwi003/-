## Phase 2. 비회원 학생 음성 제출 플로우

- [ ] 홈 화면에 비회원 업로드 CTA 배치
- [ ] 음성 파일 업로드 UI (드래그 앤 드롭 + 버튼)
  - [ ] 지원 포맷 검증 (MP3, WAV, M4A, OGG)
  - [ ] 파일 크기 제한 검증 (최대 50MB)
- [ ] 요구사항 텍스트 입력 폼 (선택 항목)
  - [ ] 예: "고음 처리가 불안정합니다", "목표 곡: OOO"
- [ ] 이메일 입력 (결과 수령용, 선택 사항)
- [ ] 업로드 완료 후 진행 상태 표시 (분석 중 UI)
- [ ] 분석 결과 열람 페이지 (비회원용 1회성 링크)
- [ ] 결과 페이지 하단 회원가입 유도 CTA
- [ ] 비회원 업로드 파일 24시간 자동 삭제 스케줄러

## Phase 3. AI 보컬 분석 기능

### 3-1. 분석 API 연동
- [ ] 음성 분석 API 선정 및 계약 (자체 개발 or 외부 API)
- [ ] 음정 (Pitch Accuracy) 분석 연동
- [ ] 박자 (Rhythm / Timing) 분석 연동
- [ ] 성량 (Volume Dynamics) 분석 연동
- [ ] 음색 (Timbre / Tone Quality) 분석 연동

### 3-2. 분석 리포트 UI
- [ ] 분석 결과 시각화 (레이더 차트 / 바 차트)
- [ ] 항목별 점수 및 텍스트 피드백 표시
- [ ] 분석 리포트 PDF 다운로드 (회원 전용)
- [ ] 회원 학생: 분석 히스토리 저장 및 열람
